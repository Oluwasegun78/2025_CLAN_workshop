# 2025_CLAN_workshop
This repository contains materials for the 2025 CLAN workshop, including presentation slides, datasets, code examples, and supporting documents. The workshop introduces participants to key concepts and provides hands-on exercises on corpus compilation and analysis.
Download Praat via https://praat.org/ [ensure you download the specification for your computer.]
Download R via https://cran.r-project.org/
Download R Studio via https://posit.co/downloads/
